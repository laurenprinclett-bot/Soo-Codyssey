# 📌 미션 1. LLM 모델 선정

---

## 🎯 1. 미션 주제

> 특정 기업 및 산업에 대한 **기사 또는 보고서**를 바탕으로 정확하고 핵심적인 내용을 정리·전달하고,  
> 이를 기반으로 아래 세 가지 정보를 제공하는 **LLM 프롬프트 엔지니어링** 진행하기.

| # | 제공 정보 |
|---|-----------|
| 1 | 🏢 연관 있는 산업 및 기업 정보 |
| 2 | 👀 앞으로 눈여겨봐야 할 주식 및 기업 정보 |
| 3 | 💰 매수를 권장하는 종목 정보 |

---

## 🧪 2. LLM 모델 선정 테스트

### 📋 2-1. 테스트 방법

특정 기업·산업 관련 기사 및 보고서 **최소 3개 이상**을 바탕으로,  
아래 두 가지 역량을 중점 평가하여 최적 모델을 선정한다.

- 📊 누가 더 **정확하고 핵심적인 정보**를 잘 정리하여 전달하는가?
- 🔮 이번 주 특정 이슈를 누가 더 **정확하게 추론**하는가?

---

### 📏 2-2. 평가 기준
| # | 평가 항목 |
|---|-----------|
| 1 | 연관 기업들을 **중요도 순**으로 제공하는가? |
| 2 | 기업의 **포지셔닝**에 대한 상세한 설명이 있는가? |
| 3 | 초보 개인 투자자를 위한 **부연 설명**을 제공하는가? |
| 4 | 요약된 정보가 **핵심 내용** 위주인가? |
| 5 | 예측된 주가가 비교적 **정확한가?** _(D+3일 기준 평가)_ |
| 6 | 정보 제공 **틀이 깔끔**하고 가독성이 좋은가? |
| 7 | _(주관적)_ 제공된 정보가 **내 포트폴리오 입맛**에 맞는가? |

---

### 💬 2-3. 공통 테스트 프롬프트

> 아래는 각 모델에 **동일하게 주입한 사용자 입력 프롬프트**이다.

### 🤖 2-4. 학습 환경

> 해당 테스트는 모두 **코디세이 사이트 내 학습 네이토** 를 통해 진행했다.

---

#### 🧑‍💻 [User Input Prompt]

```
나는 (초보) 개인 투자자야.
국장, 미장 상관 없이 모두 투자를 진행 중이고
전공 지식이나 관련된 분야에 대해 종사하고 있지 않기 때문에
기사나 보고서에 나오는 내용들을 모두 "잘" 알지 못해서
따로 검색하거나 용어, 산업에 대해 공부도 진행 중이야.

그런 나에게 중요한 건 아무래도 최신 정보일거야.
뒤쳐지면 안되고 내가 관심 있는 분야의 이슈들을 파악하고 있어야
매수, 매도에 큰 도움이 될테니까.
또한 정확한 정보는 물론 미래지향적 기사들을 토대로
성장할 섹터 및 기업을 발굴하는 것도 중요한 능력이 될거야.

따라서 너를 시험해보고자 해.
내가 매번 많은 기사, 보고서를 직접 읽는 건 아무래도 부담이 있으니
너를 활용해서 내가 선택한 몇몇 기사/보고서를
(제목을 보고 흥미 있는 것들) 요약해서 나에게 중요한 정보들을
알차게 제공해줄 수 있는 모델을 선정해서 강화 학습을 진행할거야.

너의 미션은 다음과 같아.
내가 첨부해준 기사나 실적 발표, 기업 관련 보고서를 토대로,

1. 핵심 정보를 요약하고
2. 연관 있는 산업, 기업, 이슈 정보를 제공해주고
3. 앞으로 눈여겨봐야 할 주식, 기업 정보 및
   핵심적인 연관 기업에 대해 주가/이슈 예측을 제공해줘.

개수 한정은 없고 정보 제공 틀도 없어.
네가 편한대로, 해석한대로 제공해주면 돼.
```

## 📊 3. 모델별 테스트 결과

### 🔵 테스터 모델 1 — Gemini 2.5 Pro

| 항목 | 평가 내용 |
|------|-----------|
| 기준점 설정 | 첫 번째 결과물이므로 비교 기준 **(Baseline)** 이 됨 |
| 정보 구조 | ✅ 핵심 요약을 **What / Why / So What** 관점으로 제공 |
| 연관 기업 | ⚠️ 설명이 있으나 **다소 부족** |
| 정보 충분성 | ❌ 전반적으로 **충분하다고 느껴지지 않음** |

---

### 🟠 테스터 모델 2 — Claude Sonnet 4.6

| 항목 | 평가 내용 |
|------|-----------|
| 정보 제공 틀 | ✅ **깔끔하고 명확**, 분석 전 이해를 돕는 사전 정보 제공 |
| 수치 해석 | ✅ 수치에 대한 **명확한 의미와 맥락** 제공 |
| 리스크 분석 | ✅ 실제 시장에서 영향받는 **사소한 요소까지** 현실적으로 제공 |
| 정보 구조 | ✅ **What / Why / So What** 관점의 핵심 요약 제공 |

> 💡 전반적으로 실제 **주식 분석 노트**와 같은 퀄리티를 보여줌.

---

### 🔴 테스터 모델 3 — GPT-4o

| 항목 | 평가 내용 |
|------|-----------|
| 요약 밀도 | ⚠️ 주어진 분량 내 모든 정보를 담으려는 시도 _(나름의 장점)_ |
| 과도한 압축 | ❌ **지나친 요약**으로 문맥 없이 이해하기 어려운 수준 |
| 연관 기업 | ❌ 기업명만 나열, **"왜" 주목해야 하는지 설명 부재** |

## 🏁 4. 결론 및 최종 선정

### ⚠️ 공통 한계점

> 세 모델 모두 **주가 예측 파트가 부실**하며,  
> 적극적인 전략·제안보다는 팩트 기반 전달에 그치는 경향이 있음.  
> 이로 인해 모델 간 유의미한 비교점을 찾기가 다소 어려웠음.

---

### ✅ 최종 선정 근거

**Claude Sonnet 4.6** 이 아래 항목에서 두드러진 우위를 보임:

1. 🗺️ **연관 기업 지도**를 상세하고 체계적으로 설명
2. 🙋 초보 개인 투자자가 **손쉽게 이해** 가능한 정보 제공
3. 🗂️ 정보 제공 **틀이 깔끔하고 명확**
4. 🎯 제공 내용이 **핵심 정보 위주**로 구성
5. 💼 연관 기업 및 이슈가 **내 포트폴리오 성향**에 잘 부합
