**IPO 없이 상장할 때**

- 예) 카카오, 크래프톤

**2️⃣ 직상장 (Direct Listing)**
- 공모 없이 기존 주주 주식을 바로 거래소에 올림
- 새 돈 안 들어옴, 기존 주주만 팔 수 있음
- 예) 스포티파이, 슬랙

**3️⃣ SPAC 합병**
- 이미 상장된 빈 껍데기 회사와 합병해서 우회 상장
- 청약 과정 생략 가능
- 예) 버진갤럭틱

**## 가치가 시장에 의해 형성된다는 문제**

**### 시장 가격 ≠ 내재 가치**

```

= 미래 현금흐름의 현재 가치

시장 가격 (Market Price)
= 현재 사람들이 지불하려는 금액
= 감정 + 기대 + 공포 + 탐욕의 합산
```

> 우주산업 주가는 현재 **"내재 가치"가 아닌**
> **"집단적 기대감의 가격"** 이에요

**이게 왜 위험하냐면**

```
기대감 유지 조건
├── 머스크가 계속 신뢰를 유지해야 함
├── 기술적 성과가 계속 나와야 함
├── 경쟁자가 치고 올라오면 안 됨
└── 거시경제가 위험자산을 허용해야 함

이 중 하나라도 깨지면?
→ 가격 형성의 근거가 사라짐
→ 급격한 재평가 (= 폭락)
```
